Before you download on Windows

Windows and your browser may flag Prism as unrecognized. This is normal for new apps that haven't built up a download history yet — not a sign that anything is wrong.

  • 1 Browser warning: If you see "This file may be harmful", click the ⋮ menu next to the file → Keep or Keep anyway.
  • 2 Windows SmartScreen: If a blue screen appears saying "Windows protected your PC", click More info → Run anyway.
  • 3 Install normally — Prism will appear in your Start menu and desktop.

🔒 Prism is fully open source. You can read every line of code on GitHub.

Installing Prism on Mac

Since Prism isn't distributed through the App Store, macOS will ask you to confirm it's safe to open. Here's how to install it in a few steps.

  • 1 Download and unzip the file. Double-click the .zip and a Prism.app file will appear.
  • 2 Drag Prism.app to your Applications folder.
  • 3 Right-click Prism.app → click Open → click Open again in the dialog. You only need to do this the first time.
  • 4 If you don't see the Open option: go to System Settings → Privacy & Security and click Open Anyway next to Prism.
