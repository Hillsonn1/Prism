    AI Features AI

    Prism works great on its own. Add an Anthropic API key to unlock: auto-categorization on import, smart merchant identification, and spending insights.

    Get your key at console.anthropic.com → API Keys. Stored only on your computer, never shared.

    Cards

    Rename cards imported from your statements. Changes apply to all matching transactions.

    Location

    Tell the app where you spend most. This localizes AI web searches so it can identify local businesses — especially useful outside the US. Format: City, Country (e.g. Jerusalem, Israel or New York, US).

    Clean Up Existing Transactions

    Remove Payments
    Deletes bill payments, autopay, and bank transfers that slipped through on import.
    Normalize Names
    Strips transaction codes, store numbers, and payment processor prefixes from merchant names.
    Smart Deduplicate AI
    Merges duplicate merchants with slight spelling differences, truncations, or location suffixes. One fast AI call.
    Auto-categorize AI
    Uses AI to categorize anything still uncategorized. Requires an API key.
